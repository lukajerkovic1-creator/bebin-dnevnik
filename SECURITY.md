# Sigurnost

Sigurnosne probleme nemojte objavljivati s korisničkim zapisima, lozinkama, keystoreom ili `.bdk` datotekama. Otvorite privatnu GitHub Security Advisory prijavu u repozitoriju i opišite korake reprodukcije s potpuno izmišljenim podacima.

Podržana je najnovija objavljena verzija. Gubitak Android Keystore ključa nije moguće zaobići; za oporavak je potrebna prethodno izvezena sigurnosna kopija i njezina lozinka.
